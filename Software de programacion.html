Lección 1
Software de programación por bloques

El software Conjunto de programas que permiten realizar tareas mediante un computador. de programación por bloques está diseñado para facilitar el aprendizaje de la lógica computacional, permitiendo crear programas sin necesidad de escribir código textual.

En este tipo de software, las instrucciones se representan mediante bloques Elementos gráficos que representan acciones o instrucciones. que se conectan entre sí, ayudando a comprender cómo se estructuran los programas.

Scratch

Entorno de programación Scratch

Scratch es un entorno de programación visual orientado al aprendizaje. Permite crear historias, animaciones y juegos utilizando bloques que controlan personajes llamados objetos Elementos gráficos que pueden ejecutar acciones dentro de un programa. .

Blockly

Lenguaje visual Blockly

Blockly es una herramienta de programación por bloques que permite organizar instrucciones de forma visual. Es utilizada como base en múltiples plataformas educativas para enseñar cómo se construyen los programas paso a paso.

Tinkercad

Programación por bloques en Tinkercad

Tinkercad permite aplicar la programación por bloques en proyectos con componentes electrónicos. A través de bloques se controlan luces, motores y otros dispositivos, fortaleciendo la relación entre programación y tecnología.

MakeCode Micro:bit

Entorno MakeCode Micro:bit

MakeCode Micro:bit es un entorno que permite programar una placa electrónica mediante bloques. En él se utilizan eventos Acciones que inician la ejecución de un programa. como presionar botones o detectar movimientos.

Tynker

Plataforma educativa Tynker

Tynker es una plataforma educativa que permite crear programas interactivos, promoviendo el pensamiento lógico Capacidad para razonar y resolver problemas de forma ordenada. a través de retos y proyectos guiados.

El uso de diferentes programas de programación por bloques permite a los estudiantes comprender conceptos fundamentales de la programación de manera progresiva, visual y significativa.
