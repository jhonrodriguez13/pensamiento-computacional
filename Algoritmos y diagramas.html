Lección 1
Algoritmos

El ser humano puede pensar de forma lógica y organizar acciones para resolver problemas y alcanzar objetivos.

Estas acciones pueden convertirse en instrucciones claras que los computadores ejecutan para realizar tareas, las cuales se conocen como algoritmos y se expresan mediante lenguajes de programación.

¿Qué es un algoritmo?

El lavado de manos es un ejemplo de algoritmo en la vida diaria, pues consiste en una serie de instrucciones que se realizan en un orden determinado para lograr manos limpias.

Un algoritmo es un conjunto ordenado, lógico y finito de pasos que permiten resolver un problema o alcanzar un objetivo específico.

Todo algoritmo debe cumplir ciertas características fundamentales: debe ser preciso, indicando claramente cada paso; debe ser definido, de manera que al ejecutarse varias veces produzca siempre el mismo resultado; y debe ser finito, es decir, debe tener un inicio y un final bien definidos.

Para elaborar un algoritmo correctamente se recomienda tener en cuenta los siguientes aspectos:

  • Identificar claramente el problema que se desea resolver.
  • Definir un objetivo que permita evaluar si el problema fue solucionado.
  • Diseñar una secuencia lógica de pasos para alcanzar el objetivo.
  • Probar el algoritmo y verificar que los resultados sean correctos.

Ejemplo 1

Elabore un algoritmo que permita encender un computador y acceder al escritorio.

Objetivo: Encender el computador y acceder al escritorio.

Inicio

  1. Presionar el botón de encendido del computador
  2. Si el sistema solicita contraseña entonces ingresarla correctamente.
  3. Si no solicita contraseña entonces esperar a que cargue el sistema operativo
  4. Visualizar el escritorio del computador

Fin

Ejemplo 2

Plantee un algoritmo que permita enviar un mensaje desde un teléfono móvil.

Objetivo: Enviar un mensaje desde un teléfono móvil.

Inicio

  1. Encender el teléfono móvil
  2. Si el teléfono tiene bloqueo entonces desbloquearlo
  3. Si no tiene bloqueo entonces abrir la aplicación de mensajes
  4. Escribir y enviar el mensaje

Fin

Existen diferentes tipos de algoritmos según la forma en que se organizan las instrucciones. Los algoritmos secuenciales siguen una serie de pasos ordenados que se ejecutan uno tras otro sin interrupciones. Los algoritmos condicionales incluyen decisiones, permitiendo que el proceso cambie según se cumpla o no una condición. Por su parte, los algoritmos repetitivos ejecutan uno o varios pasos varias veces hasta que se alcanza un objetivo determinado. Estos tipos de algoritmos están presentes tanto en situaciones cotidianas como en los programas informáticos utilizados por los computadores.
